# Local Canvas 前端设计规格

> 2026-06-09 | 脑暴产出 | 待用户审核

---

## 一、定位与布局

### 双场景定位
- **日常办公自动化**：拖拽搭工作流，串联 AI 模型/API/知识库
- **开发者调试**：可视化编排 skill 调用链，实时日志与配置

### 整体布局
三栏固定布局，从左到右：
- **左侧**：资源面板（w-56，约 224px）
- **中央**：画布工作区（弹性宽度）
- **右侧**：AI 对话框（默认 400px，可拖拽分隔条调节 300-500px）

顶部 40px 工具栏，底部无状态栏。

### 窗口框架
- 自定义无边框 Electron 窗口
- 自绘窗口控制按钮（最小化/最大化/关闭）
- 导航栏与拖拽区域合一

---

## 二、色彩与视觉语言

| 类别 | 值 |
|------|-----|
| 底色 | surface-850 ~ surface-950 深灰黑系列 |
| 主色调 | amber-400 / amber-500 琥珀金 |
| 辅色 | 区域色区分（Skill 翠绿、知识库紫、输出琥珀） |
| 圆角 | 12-16px 大圆角 |
| 字体 | 等宽优先，终端 hacker 气质 |
| 滚动条 | 5px 琥珀色半透明，hover 加深 |
| 动画 | 克制，200ms ease，仅必要处使用 |

---

## 三、顶部工具栏（h-10）

纯图标 + tooltip（悬停显示名称及快捷键），无文字标签。分组：

| 分组 | 按钮 | 图标 | 快捷键 |
|------|------|------|--------|
| 文件 | 加载工作流 | FolderOpen | Ctrl+O |
| | 保存 | Save | Ctrl+S |
| | 导出 JSON | Download | Ctrl+E |
| 编辑 | 撤销 | Undo2 | Ctrl+Z |
| | 重做 | Redo2 | Ctrl+Y |
| | 清空画布 | Trash2 | — |
| 运行 | 运行工作流 | Play | F5 |
| | 触发器设置 | Timer | — |

按钮样式：hover 微提亮 + scale(1.03)，active scale(0.97)。

---

## 四、左侧资源面板（w-56）

### 切换方式
顶部四个分类标签，点击切换，一次显示一个类别：
- FlaskConical Skills
- Globe 知识库
- Brain 模型
- Plug API

### 卡片样式
- 较大卡片，显示图标 + 名称 + 完整描述 + 版本/类型标签
- hover 上浮 2px + 边框变亮
- 拖拽反馈：原卡片 50% 透明度，画布显示虚线引导区 + 放置后扩散动画

---

## 五、中央画布

### 自定义节点
三种类型，每种 12px 圆角卡片：

| 类型 | 图标 | 边框色 | 信息 |
|------|------|--------|------|
| Skill | FlaskConical | 翠绿 | 名称 + 描述 + 状态标签 |
| 知识库 | Globe | 紫色 | 名称 + 描述 + 状态标签 |
| 输出 | Send | 琥珀 | 名称 + 状态标签 |

### 连线
- 虚线/波浪线 + 末端箭头
- 颜色随源节点类型
- 流动动画
- 连线完成弹出映射确认弹窗

### 节点交互
- 单击选中 = 边框高亮
- 双击 = 浮动配置弹窗
- Delete/Backspace = 硬确认弹窗后删除
- 拖拽移动、连线、拖入添加

### 空画布
展示 2-3 个角色化模板卡片：场景插图 + "适合 xxx 场景"标签 + 节点数，点击一键导入。

---

## 六、节点配置弹窗

### 触发
双击节点

### 样式
macOS popover 风格：12px 圆角卡片 + 顶部三角指向节点 + 轻微阴影

### 内容
- 节点名称（可编辑）
- 节点类型
- 模型选择下拉
- API Key 输入（密码框）
- 参数 JSON 编辑器（等宽 textarea）
- 输入/输出 schema 预览
- 连线映射预览
- 实时状态指示
- 保存 / 删除按钮

---

## 七、右侧 AI 对话框（400px 默认）

### 顶部
- 可编辑对话标题
- 历史对话列表下拉
- 新建对话按钮 Plus
- 模型选择下拉
- 清空对话按钮 Trash2

### 消息气泡
- 每条消息前有角色图标（User=用户图标，AI=Sparkles）
- Markdown 渲染（代码块、链接、列表）
- 用户消息靠右 + 琥珀色底
- AI 消息靠左 + 深灰卡片底
- 系统消息居中 + 斜体小字

### 输入区
- 多行自适应 textarea（高度 40-120px）
- Shift+Enter 换行，Enter 发送
- 发送按钮 Send（无文字）
- 空状态展示快捷提示词按钮

---

## 八、浮动日志窗口

### 触发
点击运行后自动弹出

### 样式
终端风格：
- 纯黑底 #0a0a0a
- 等宽字体
- 琥珀色 amber-400 日志文字
- 彩色圆点区分 level（info/绿、warn/琥珀、error/红）
- 清屏按钮 Eraser
- 导出日志按钮 Download
- 可拖拽、可调整大小

---

## 九、设置页面

三个标签：模型管理 / API Key / 知识库

### 表单样式
- 点击"添加" = 右侧滑出面板（w-80）
- 表单字段垂直排列
- 保存/取消按钮底部固定
- 主列表左侧显示，支持 hover 显示删除按钮

---

## 十、连线映射弹窗

### 触发
两条节点连线完成时

### 样式
居中模态弹窗，400px 宽：
- 顶部显示源节点 = 目标节点流向
- 源输出字段下拉选择
- 目标输入字段下拉选择
- 取消 / 确认映射按钮

## 十二、技术实现要点

- Electron 主进程 CJS（main.cjs）
- React 渲染进程，Vite 构建
- 状态管理：Zustand
- 画布：@xyflow/react v12
- UI：TailwindCSS + lucide-react 图标
- Markdown：react-markdown
- 通知：react-hot-toast
- HTTP：axios = 代理到后端 localhost:3001
- WebSocket：原生 WebSocket = ws://localhost:3001/ws
